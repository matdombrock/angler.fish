# ROD
Change your fish shell prompt.

> [!NOTE]
> This file is intended to be sourced (`source rod.fish`) and not executed directly.
> 
> This is required for setting the shell prompt

## Sourcing

To use the `:rod` command you must first source this file: 

```sh
source rod.fish
:rod minimal
```

## Usage

### List
To list available prompt styles run:
```sh
:rod list
```

### Change
To change your prompt style run:
```sh
:rod [style]
```

### Example
For example:
```sh
:rod minimal
```
