# LLM

## Setup

Create an `llm.cfg.fish` file in this dir. 

You can use the `llm.cfg-example.fish` script as a base. 

## Use

```fish
# Get a shell command
:llmsh list the files in my home dir
# Ask a quick question
:llm "whats up?"
```

*Note: You can usually get away without quoting your prompt, but if you run into issues with special chars, try quoting it.*
